 ## Python 🐍


> La extencion del archivo debe ser siempre **.py** de manera que se vea así: [Script.py](./script.py)

---

>> **Nota:** Un **script** es un conjunto de instrucciones o código, generalmente sencillo y no compilado, escrito en un lenguaje de programación para automatizar tareas, manipular datos o controlar aplicaciones

---

### Tipos de datos

- Entero (int)
- Reales (float)
- cadenas (string)
- Verdadero/Falso (bool)

---

### Imprimir o mostrar 

- Print -> Escribir 

```python

print("Por favor ingrese su nombre:")

```
---

### Definición de variables

- En python no se declaran variables de manera compilada. 

>> **Nota:** se usara snake_case para las variables en python

```python

nombre_apostador = "Pedro Pérez"
numero_magico = 7
promedio_notas = 3.8

```
#### Imprimir variables

```python

print(nombre_apostador)

```
